    TS是什么？

    TypeScript 是 JavaScript 的超集,TypeScript 也是 JavaScript ES6 的超集

    为什么使用TS

    • 提高开发效率：TS在vscode，webstorm等ide下有很好的智能提醒支持，平时项目中使用的第三方框架库都有TS类型声明。
    • 提高维护性：TS有强类型约束和静态检查，可以使不同维护人员来进行代码更新替换也不会有太大差异，提高可维护性。
    • 质量高：很多时候项目bug都是调用方和被调用方（如组件模块间的协作，接口或函数的调用）的数据不匹配所引起的，由于TS编译期的静态检查，让bug率大大降低。
